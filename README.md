This program does the following:
```
+========================================================================================+ 
|                             HELP - CSV Person Data Manager                             |
+========================================================================================+
| This program allows you to manage and query a dataset of persons using CSV files.      |
| Each person has the following fields: CPF, RG, Name, Birthdate, and City.              |
| The expected CSV format is: CPF,RG,Name,Birthdate,City (no headers).                   |
| Example: 012.345.678-90;9876543210;John Doe;23/12/1988;Westford.                       |
+----------------------------------------------------------------------------------------+
| OPTIONS:                                                                               |
| [1] Consult by.           - Access submenu to search by CPF, Name, or Birthdate range. |
| [2] Load.                 - Access submenu to load from a file or directory.           |
| [3] Update or add.        - Updates or adds a new person manually.                     |
| [4] Save to file.         - Saves the current dataset to a CSV file.                   |
| [5] Delete.               - Access submenu to delete a person by CPF or everything.    |
| [6] Print.                - Access submenu to print size, person, or tree details.     |
| [7] Help.                 - Display this help message.                                 |
| [8] Clear console.        - Clears the screen.                                         |
| [9] Exit.                 - Exit the program.                                          |
| [10] Verbose mode.        - Will print CPFs that were updated when loading a file.     |
+----------------------------------------------------------------------------------------+
| INPUT FORMATS:                                                                         |
| CPF                       - Only numeric or in the format ###.###.###-##.              |
| RG                        - Only numeric.                                              |
| Name                      - Alphanumeric, case-insensitive.                            |
| Birthdate                 - Format: DD/MM/YYYY.                                        |
+========================================================================================+
|                                  Program by Jean Rehr                                  |
+========================================================================================+
```
